# Fundamentals of Artificial Intelligence Labs

This repository contains laboratory reports and practical projects completed as part of the Artificial Intelligence course within the Cybersecurity specialization.

The work focuses on the application of machine learning, deep learning, and reinforcement learning techniques to data analysis, prediction, classification, clustering, and intelligent decision-making. These labs provide hands-on experience with modern AI algorithms and the tools commonly used in data science and cybersecurity research.
